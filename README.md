## 📖 About Community Health

**_Community Health_** is the organization-level repository through which _GitHub_ sources default community health files — issue and pull request templates, code of conduct, contributing guide, security policy, support policy, and funding configuration — for every repository under [@bilbilak](https://github.com/bilbilak) that does not provide its own.
